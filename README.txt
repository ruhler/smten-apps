Smten Applications
==================
Richard Uhler <ruhler@csail.mit.edu>
June 10, 2014

A collection of applications for smten.

mini:: A collection of small examples.
  Includes a bounded model checker and string constraint solver.
nqueens:: An nqueens puzzle solver.
saiger:: An AIGER model checker for property verification.
shampi:: A reimplementation of Hampi, a string constraint solver.
ssketch:: A reimplementation of Sketch, a tool for program synthesis.
sudoku:: A 9x9 sudoku puzzle solver.
